Winamp plugin "gen_tbar"

About

This is a free plugin for Winamp (versions 2 and 5) that displays customizable data on the caption of the active window. Here are some of the options:

  • position in the playlist
  • artist name
  • song title
  • remaining time

Options

You can change the plugin's default options ( Winamp preferences -> Plug-ins -> General Purpose -> Titlebar Info x.xx (gen_tbar_2.dll) -> Configure). All settings (except for Caption of window...) take effect at once, so you don't have to close the dialog window to see the changes.

Question mark (?): Display help for the control you click on.
Plugin [en|dis]abled: Enable or disable the plugin.
Show remaining time: Display the remaining time of the song instead of the current time.
Show playback status: Display blinking [Pause] and [Stop] on the caption.
[Font name]: Select the font that should be used.
Horizontal shift: Shift the font horizontally on the caption.
Vertical shift: Shift the font vertically on the caption.
Display configuration: You can change what is displayed on your caption by using a few variables. %time% for instance will display the current or remaining time of the song, while %artist% will show the artist's name. Note that a couple of variabes can only be used in Winamp 2.90 or Winamp 5 and also require ID3 information to be present in the current song.
Reset: Click here if you messed up the display configuration and want to restore the default setting.
Window captions [...]: If the text you specified here appears somewhere in a caption, the plugin won't display data on it. You can specify up to ten different strings if you separate them with semicolons: string1;string2;string3. Comparision is case sensitive. In case you experience problems with this option, just remove all text from the text box, and the data should appear on the caption again.
Ok: Apply new settings and exit.
Cancel: Exit without changes.
Help: Displays this help file.
About: About this plugin.

Uninstall

I included an uninstaller with the plugin. If you still want to uninstall manually, please follow the instructions below:

  • Close Winamp and open the subdirectory Plugins of your Winamp-Directory
  • Delete the files gen_tbar_2.dll and gen_tbar.chm

If you know how to use RegEdit, you may delete the registry key HKEY_CURRENT_USER\Software\radix\gen_tbar. Otherwise you'll better leave this alone!
